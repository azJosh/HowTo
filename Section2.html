Section 2: Download the PHP SOAP Library

This library gets product information using the Amazon Product Advertising API. The two main functions in this library are ItemSearch and ItemLookup. This library requires the PHP extentsion SOAP.

Download the Library

Go to https://aws.amazon.com/code/Product-Advertising-API/4373 and click the "Download" button. Download the latest version.



We will be using the file AmazonECS.class.php file. Save this file in the same folder as the program you will creating.
